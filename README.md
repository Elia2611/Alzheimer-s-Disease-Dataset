Estructura del repositori Alzheimer's Disease Dataset:
 - ADD.ipynb			<- entrenament i avaluament dels models 
 - Alzheimer.zip		<- carpeta amb tot el contingut necessari per generar el document LaTeX de l’informe
 - alzheimers_disease_data.csv	<- dades utilitzades
 - PrimeraAproximacio.ipynb	<- codi d'EDA i preprocessament
 - README 			<- aquest fitxer


Descripció del projecte: Aquest projecte té com a objectiu desenvolupar un model de classificació capaç de predir si un pacient presenta Alzheimer a partir de variables clíniques i biològiques. Inclou diverses etapes: preprocessament de dades, selecció de mètriques, entrenament de models, comparació de resultats i discussió.

Requeriments:
 - Python 3.10
 - pandas
 - numpy
 - matplotlib
 - seaborn
 - scikit-learn

Per executar el codi:
 1. Obrir els notebooks:
  - PrimeraAproximacio.ipynb
  - ADD.ipynb
 
 2. Executar totes les cel·les de cada fitxer.
